# Documentation

This directory contains all project documentation organized by category.

## Structure

```
docs/
├── deployment/      # Server deployment and transfer guides
├── data/            # Dataset documentation and notes
├── models/          # Model architecture and design documentation
├── baseline/        # Baseline model documentation
├── TROUBLESHOOTING.md  # Common issues and solutions
└── Thema.pdf        # Thesis document
```

## Categories

### deployment/
Server deployment and code transfer documentation:
- `SERVER_DEPLOYMENT.md` - Complete server deployment guide for bwUniCluster 3.0
- `SERVER_EXPORT.md` - Server export and training guide
- `SERVER_QUICKSTART.md` - Quick start guide for server usage
- `DEPLOYMENT_TRANSFER.md` - Code transfer and deployment procedures

### data/
Dataset documentation and notes:
- `mimic_iv_ecg_notes.md` - Important limitations and usage notes for MIMIC-IV-ECG
- `MIMIC_IV_SCAN.md` - MIMIC-IV database scan and analysis

### models/
Model architecture and design:
- `model_architecture_design.md` - Methodical design for CNN, LSTM, Hybrid, Transformer models

### baseline/
Baseline model documentation:
- `sofa_score_slide.md` - SOFA score documentation and slides
