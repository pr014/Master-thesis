# Master Thesis

ECG-based Length of Stay Prediction in ICU using Deep Learning

## Project Structure

- `configs/` - Configuration files for different datasets and models
- `scripts/` - Training scripts and cluster job files
- `src/` - Source code for models, data loading, and training
- `data/` - Dataset directories (not tracked in git)
- `outputs/` - Training outputs and checkpoints (not tracked in git)

## Datasets

- `icu_24h` - ICU ECGs from first 24 hours
- `all_icu_ecgs` - All available ICU ECGs

## Requirements

See `requirements.txt` for dependencies.
